## Installation

```sh
pip install -r requirements.txt
```
```
pip install Flask-SQLAlchemy
```
```
pip install Flask-WTF
```
```
pip install flask-cors
```
## How to Run

```
export FLASK_APP=app.py
export FLASK_ENV=development
```
```
flask run
```
